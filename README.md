## Kurzinfo zum Rad

- **Modell:** Prophete Cargo Plus 22.ETL.10  
- **Antrieb:** AEG ComfortDrive Mittelmotor (250 W, 100 Nm, Unterstützung bis 25 km/h)  
- **Technische Basis:** Bafang (u. a. an Fehlercode Error 21 erkennbar). Bafang-Ersatzteile können kompatibel sein.

Garantie und rechtliche Fragen (z. B. 25-km/h-Begrenzung) bleiben bei Prophete bzw. Fachwerkstatt. Bei Unsicherheit immer Kundendienst oder autorisierte Werkstatt nutzen.
